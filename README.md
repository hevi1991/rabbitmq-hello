A "Hello World" demo of RabbitMQ

producer `send.js`

consumer `receive.js`


## Installation

> RabbitMQ server is needed. Install it by docker:
> 
> ```
> docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.13-management
> ```


```
npm install
```

## Usage

```
npm start
```
